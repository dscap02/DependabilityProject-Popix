Pop!x – JaCoCo Code Coverage Evolution

This page documents the evolution of code coverage for the Pop!x project. Coverage reports are generated using JaCoCo and published as static artifacts via GitHub Pages.

Reports are grouped by testing milestones to highlight the impact of different testing strategies (unit and integration testing).

Coverage Reports

  • Baseline
    Initial coverage measured before introducing additional tests.
  • After Unit Tests
    Coverage after adding unit tests for the service layer.
  • After Integration Tests
    Coverage after introducing integration tests on the persistence layer.
  • Final Coverage
    Final coverage after completing all planned testing activities.

Notes

Coverage analysis includes only project-owned code and excludes third-party libraries to ensure meaningful and academically valid metrics.
